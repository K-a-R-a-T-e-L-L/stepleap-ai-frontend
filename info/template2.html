Паспорт навыков (Skill Passport)

Не «анкета», а профиль с доказательствами: мини-кейсы → артефакты → рост совпадений по вакансиям

Компас интересов

S · E · I
Социальное + предпринимательское + исследовательское. Рекомендованы треки: HR, продажи/аккаунтинг, аналитика.
Сильная зона: коммуникация
Зона роста: системность

Карта пути (Карьера как «метро / орбиты»)

Каждая «станция» — шаг, который реально приближает к роли (подработка → стажировка → первая позиция)
Трек выбран: HR / рекрутинг
①
Станция: «Понимание вакансии» Разобрать требования, выделить компетенции, составить «матрицу соответствия».
②
Станция: «Скрининг резюме» Научиться быстро отсеивать нерелевантные резюме. Эпизод-практика на 10 примерах.
③
Станция: «Первичное интервью» Сценарий звонка/созвона + этика вопросов + фиксация результата в шаблон.
④
Станция: «Портфолио стажера» Собрать 3 Proofs: чеклисты, шаблоны, мини-отчеты и выгрузить в резюме.

Эпизоды-профпробы

Симуляции + мини-кейсы → доказательства навыков → рост совпадений по вакансиям
Try-Out

Эпизод: «День HR-стажера»

7 минут · 3 сцены · результат: чеклист + мини-отчет
уровень: старт
Сцены: понять вакансию → выбрать 3 кандидатов → написать 1 приглашение в корректном тоне.
Skill Sprint

Спринт: «Excel/таблицы для работы»

10 минут · 1 упражнение · результат: шаблон таблицы кандидатов
эффект: + вакансии
Собрать таблицу: кандидат → статус → следующий шаг → дата → заметки. Добавить фильтры и приоритеты.
